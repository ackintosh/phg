# PHG - PHP: Hypertext Preprocessor written in Go

This project is designed for studing golang and php.

## Installation

```
$ git clone git@github.com:ackintosh/phg.git
$ cd phg
$ go build
```

## Usage

```
$ cat sample.php
<?php
echo "Hello, World!";
```

```
$ ./phg sample.php
Hello, World!
```
